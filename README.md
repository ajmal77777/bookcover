# Book Cover – Fundamentals of Web Application Development

## Files
- `index.html` — Main HTML page
- `style.css`  — Stylesheet
- `author-photo.jpg` — **Add your own author photo here** (rename your image to this)

## How to Run
1. Place your author photo in this folder and rename it to `author-photo.jpg`
2. Open `index.html` in any web browser — no server needed.

## Customising
- To change the author photo: replace `author-photo.jpg` with your image file,
  or update the `src` attribute in `index.html` to match your filename.
- To adjust the price, publisher name, or any text: edit `index.html` directly.
